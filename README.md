# INFINITE_GOLD

## 概要
鉱石を採掘してお金を増やす、クリッカー系ゲームです。
様々な強化要素によって、レアな鉱石の出現確率を上昇させたり、採掘力をあげることなどができます。<br>
pygameを用いて開発しました。

-----

## 制作背景
初めてのゲーム制作なので、自分の好きなジャンルであるクリッカー系ゲームで、ランダムな要素もありつつ、戦略性も持たせるゲームを作りたいという思いで制作しました。<br>
鉱石やお金などそれぞれの変数に複数の使用方法があり、プレイヤーが遊び方を選択できるようにしています。<br>
また、実績システムによってプレイヤーが目標を設定しやすいようにしました。

-----

## 技術要素
確率ベースでレアな鉱石が出現するようになっています。
確率は強化することができ、その確率変動にはS字カーブ型の有理関数を使用しています。<br>
また、自動で鉱石を採掘してくれるキャラクターを雇うことができるようになっています。
このキャラクターは自分から最も近い鉱石（当てはまる鉱石が複数ある場合はレア度が高い順、レア度も同じ場合は右下左上の順）を採掘するようにプログラムしています。
